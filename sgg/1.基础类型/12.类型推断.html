# 类型推断

类型推断是指 TS 会在没有明确的指定类型的时候推测出一个类型

共有两种情况：

  1. 定义变量时赋值了, 推断为对应的类型.
  2. 定义变量时没有赋值, 推断为 any 类型
// 情况1：自动推断为number类型
let num = 10;
num = "哈哈"; //不能将类型“string”分配给类型“number”。

// 情况2：自动推断为any类型
let num;
num = 10;
num = "哈哈";
