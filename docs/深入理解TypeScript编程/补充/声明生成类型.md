# 声明生成类型

TS中有些声明语句在声明一个变量的同时会声明以该变量名为名的类型，而另一些声明语句则只是简单的声明该变量。

声明语句声明的类型与基础类型别无二致，可以用作类型注解（曾为此大惑不解）

总结如下：

| 声明语句        | 是否生成类型 | 是否生成值 |
| --------------- | ------------ | ---------- |
| class           | 是           | 是         |
| enum            | 是           | 是         |
| interface       | 是           | 否         |
| type            | 是           | 否         |
| const、let、var | 否           | 是         |
| namespace       | 否           | 是         |
| function        | 否           | 是         |
